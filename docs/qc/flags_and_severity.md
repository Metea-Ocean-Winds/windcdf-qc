# Flags and Severity

## Flag Levels

| Level | Name | Description |
|-------|------|-------------|
| 0 | GOOD | Data passed all checks |
| 1 | SUSPECT | Data questionable, review recommended |
| 2 | BAD | Data failed QC, likely erroneous |
| 3 | MISSING | No data available |

## Bitmask Convention

Flags can be combined using bitmasks to indicate multiple issues.
